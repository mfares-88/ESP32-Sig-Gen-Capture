1. Library Installation & File Setup
To begin, the LVGL library must be installed and the configuration file properly located within the Arduino libraries folder.

Install LVGL Library:

Open the Arduino IDE.

Navigate to Tools > Library Manager.

Search for and install the standard lvgl library.

Prepare the Configuration File:

Navigate to the directory where the library was installed (e.g., inside Documents/Arduino/libraries/lvgl).



Locate the file named lv_conf_template.h.



Copy this file to the Main Libraries folder (the parent directory, e.g., Documents/Arduino/libraries/).



Rename the copied file from lv_conf_template.h to lv_conf.h.

2. Configuring lv_conf.h
Open the newly created lv_conf.h file in a text editor (like Notepad)  and modify the following lines to match the Guiton JC4827W543 hardware requirements.

A. Enable Configuration
Change the enablement flag at the top of the file to activate the configuration:

C

/* Set it to "1" to enable content */
#if 1 


B. Display Settings
Set the default Dots Per Inch (DPI) to 130 to match the screen density:

C

#define LV_DPI_DEF 130


C. Operating System (OS) Setup
Configure the library to use FreeRTOS, which is required for the ESP32-S3 environment:

C

/* Select an operating system to use. */
#define LV_USE_OS   LV_OS_FREERTOS


Enable direct task notifications for optimized performance:

C

#define LV_USE_FREERTOS_TASK_NOTIFY 1


D. Theme Configuration
Enable the default theme and configure its transition settings:

C

#define LV_USE_THEME_DEFAULT 1
#define LV_THEME_DEFAULT_TRANSITION_TIME 80



E. Fonts
Enable the specific font sizes required for the interface (Montserrat 10 and 14):

C

#define LV_FONT_MONTSERRAT_10 1
#define LV_FONT_MONTSERRAT_14 1



Set the system default font to Montserrat 14:

C

#define LV_FONT_DEFAULT &lv_font_montserrat_14


(Optional) Enable Persian/Hebrew fonts if required:

C

#define LV_FONT_DEJAVU_16_PERSIAN_HEBREW 1


F. Language Support (RTL)
Enable support for Right-to-Left languages and Arabic/Persian characters:

C

/* Support bidirectional texts */
#define LV_USE_BIDI 1

/* Automatically detect base direction */
#define LV_BIDI_BASE_DIR_DEF LV_BASE_DIR_AUTO

/* Enable Arabic/Persian character processing */
#define LV_USE_ARABIC_PERSIAN_CHARS 1
